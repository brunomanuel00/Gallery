# Gallery

Proyecto: Galería de Fotos
Caso de estudio: Una aplicación web que permite a los usuarios crear y administrar una galería de fotos.
Requisitos:
-Página de inicio: Crea una página de inicio que muestre las fotos del usuario en una cuadrícula.
-Agregar fotos: Permite al usuario agregar nuevas fotos a la galería.
-Ver detalles de la foto: Permite al usuario hacer clic en una foto y ver su título y descripción.
-Eliminar fotos: Permite al usuario eliminar fotos de la galería.
-Editar detalles de la foto: Permite al usuario editar el título y la descripción de las fotos en la galería.
-Guardar fotos: Guarda las fotos en el almacenamiento local del navegador para que el usuario pueda acceder a ellas más tarde.
-Diseño Responsivo: Asegúrate de que la aplicación esté diseñada para ser responsiva y accesible en diferentes dispositivos, incluyendo computadoras de escritorio, tabletas y teléfonos móviles.

Implementación:
-Página de inicio: Crea una página de inicio simple que muestre las fotos del usuario en una cuadrícula utilizando HTML y CSS.
-Agregar fotos: Crea un formulario que permita al usuario agregar una nueva foto a la galería utilizando JavaScript. Tenga en cuenta que las fotos tienen un título y una descripción.
-Ver detalles de la foto: Agrega una función que permita al usuario hacer clic en una foto y ver su título y descripción utilizando JavaScript.
-Eliminar fotos: Agrega un botón que permita al usuario eliminar fotos de la galería utilizando JavaScript.
-Editar detalles de la foto: Agrega un botón que permita al usuario editar el título y la descripción de una foto utilizando JavaScript.
-Guardar fotos: Usa el almacenamiento local del navegador para guardar las fotos del usuario para que puedan acceder a ellas más tarde utilizando JavaScript.
-Diseño Responsivo: Usa media queries y diseños flexibles para asegurarte de que la aplicación sea responsive y accesible en diferentes dispositivos, incluyendo computadoras de escritorio, tabletas y teléfonos móviles.
